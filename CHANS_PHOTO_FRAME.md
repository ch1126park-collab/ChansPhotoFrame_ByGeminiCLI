# 🖼️ 재현이를 위한 스마트 전자액자 (Chan's Photo Frame)

안드로이드 태블릿(Lenovo P11 등)에 최적화된 고성능 인텔리전트 전자액자 프로젝트입니다.

## 🚀 핵심 기능

### 1. 기본 슬라이드 쇼 (Basic Slideshow)2

* **자동 사진 스캔**: `/sdcard/Media/Photoframe/` 및 하위 폴더의 모든 이미지(JPG, PNG) 자동 인식.
* **다양한 트랜지션**: Fade, Slide(가로/세로), Scale 효과가 랜덤하게 적용되어 부드러운 화면 전환 제공.
* **방향 맞춤 필터링**: 태블릿이 가로 모드면 가로 사진만, 세로 모드면 세로 사진만 지능적으로 선별하여 재생.
* **이머시브 모드**: 상태 표시줄과 내비게이션 바를 숨겨 화면 전체를 사진으로 꽉 채움.

### 2. 지능형 추억 소환 (Smart Memories)

* **추억 여행 모드 (Related Photos)**: 사진을 **길게 누르면** 현재 사진과 관련된 사진(같은 폴더, 앞뒤 30일 이내, 혹은 같은 도시)들을 2시간 동안 집중 재생.
* **시각 효과**: 추억 여행 모드 진입 시 **부드러운 줌인(Zoom-in)** 연출 및 황금색 "추억 여행 중" 배지 표시.
* **과거 오늘 하이라이트**: 설정에서 활성화 시, 매년 오늘 날짜(월/일 일치)에 찍혔던 추억들만 선별하여 재생.

### 3. 고정밀 정보 오버레이 (Metadata Overlay)

* **재현 맞춤형 연령 표기**: 촬영일 기준 재현(2017.12.11생)의 성장 단계를 자동 계산.
  * ~2018년: `재현 D+일수`
  * 2019~2020년: `재현 XX개월`
  * 2021년~현재: `재현 X살` (한국식 나이)
* **상세 주소 및 인앱 지도**:
  * GPS 정보 기반 상세 주소 표시 (국가명 포함: 대한민국/수원시/...).
  * 주소 터치 시 앱 내에서 **90% 크기의 구글 지도 팝업**을 띄워 위치 즉시 확인.

### 4. 고성능 엔진 (Performance)

* **SQLite DB 인덱싱**: 1만 장 이상의 사진 정보를 DB화하여 검색 및 필터링 속도 극대화 (v4 스키마: 날짜/위치 인덱스 포함).
* **지능형 프리패칭 (Pre-fetching)**: 다음 사진을 백그라운드에서 미리 로드하여 전환 시 지연 시간 0초 구현.
* **대용량 캐시**: RAM의 25%를 이미지 메모리로 할당, 512MB 전용 디스크 캐시 구축.

### 5. 스마트 전력 관리 (Power Management)

* **자동 취침 예약**: 새벽 1시 ~ 6시 사이에는 가동을 멈추고 시스템 절전 모드 진입 허용.
* **상황 인지 절전**:
  * 충전기가 연결되어 있을 때만 화면 켜짐 유지.
  * 가속도 센서를 통해 태블릿이 바닥에 눕혀진 상태(보관 중)로 판단되면 즉시 대기 모드로 전환.
* **슬라이드 진행 바**: 화면 하단에 다음 사진 전환까지 남은 시간을 시각적으로 표시.

## ⚙️ 설정 및 조작

* **설정 팝업**: 화면 터치 시 나타나는 톱니바퀴 아이콘으로 진입.
  * 재생 순서(순차/랜덤), 노출 시간(3/10/20초), 하이라이트 모드 토글 가능.
  * **DB 초기화 버튼**: 파일 시스템 변화 시 수동으로 전체 다시 스캔 가능.
* **간편 제스처**:
  * 한 번 터치: 일시정지 / 재생
  * 더블 터치: 재생 모드(순차/셔플) 즉시 전환
  * 양옆 터치: 이전/다음 사진 수동 넘기기
